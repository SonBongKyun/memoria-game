플레이해줘서 고마워. 약 30분 걸려.
다 하고 나서 아래 양식에 답해서 보내주면 돼. **솔직하게** 답하는 게 가장 큰 도움이야.

---

## 시작 방법

1. zip 파일 압축 풀기
2. `MEMORIA-Demo-v0.1.exe` 더블클릭
3. (Windows Defender 경고 뜨면 "추가 정보 → 실행" 클릭)

## 조작

| 키 | 행동 |
|---|------|
| **WASD / 화살표** | 이동 |
| **Space / Enter / 좌클릭** | 대화 진행, 상호작용 |
| **Tab / M** | 기억 보관소 |
| **ESC** | 일시정지 / 메뉴 |
| **F6 / F7** | 빠른 저장 / 불러오기 |

## 끝나는 지점

화면에 **"Act I — Ash · End of Demo"** 가 뜨면 끝.

**C2.** 그 선택 *이후*에 뭔가 바뀌었다는 걸 느꼈어? (대사 변화, 화면 효과, NPC 변화 등)

**C3.** Tab 키를 누르고 **"Constellation"** 버튼은 한 번이라도 눌러봤어?

## D. 조작감 / 버그

**D1.** 게임이 **멈춘** 순간이 있었어? (대사 안 넘어감, 캐릭터 안 움직임, 화면 까만 채로 정지 등)
> 어디서 / 뭘 누르려 했는지:

**D2.** **이상하게** 보이는 비주얼이 있었어? (그림 깨짐, 글자 겹침, 색깔 이상함)
> 

**D3.** 글씨 속도 / 글자 읽는 속도 — 너무 느려 / 적당해 / 너무 빨라
